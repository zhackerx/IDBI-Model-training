# Risk Guidelines (Sample)

## Decision Bands
- Approve: strong evidence across compliance and repayment behavior.
- Conditional Approval: mixed evidence or partial document coverage.
- Reject: clear high-risk evidence including defaults and severe non-compliance.

## LLM/Judge Expectations
- Each section must cite evidence from applicant data or policy chunks.
- If evidence confidence is low, workflow must trigger corrective retrieval.
- Contradictions between section conclusions should be flagged for review.
